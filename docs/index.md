# GaasHub Documentation

Welcome to the official documentation for **GaasHub**, the ultimate data platform for modern AI.

GaasHub empowers researchers and developers by simplifying the bridge between local development and powerful remote GPU infrastructure.

## Run GPU workloads on any device without setup

With GaasHub, you can offload heavy computations, model training, and data processing to high-performance GPU clusters with just a single command. No more complex infrastructure setup, CUDA versioning issues, or expensive hardware requirements.

### Quick Start

Get started in seconds:

1. **Download & Install**: Get the GaasHub app from [gaashub.com](https://gaashub.com).
2. **Connect**: Launch the app, sign in, and click **Connect** to a GPU node.
3. **Run**: Use the `gaashub` command to run your script on remote GPUs:

```bash
gaashub run train.py
```
